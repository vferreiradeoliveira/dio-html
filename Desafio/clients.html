Clients no linguajar da Computação

São dispositivos que consomem informação de um Servidor, sendo eles Notebooks, Computadores,Smartphones e etc.

Forma de Comunicação

Em uma comunicação Web, por exemplo abrir um site o Cliente manda uma solicitação para o Servidor dos arquivos estaticos do Site e o Servidor devolve estes arquivos para o cliente o navegador interpreta esses arquivos e exibe o site.

Arquivos estaticos são:

Arquivos HTML,CSS,Javascript,Imagens,Videos etc...

Nessa Comunicação para que não ocorra sempre esse envio desses arquivos o Navegador guarda essas informações em Cache.

Navegadores

São programas criados para interpretar os arquivos HTML,CSS,Javascript,Imagens entre outros.Seguindo padrões impostos pela W3C.

Iniciou com MOSAIC,depois Netscape e atualmente existem diversos como Google Chrome,Brave,Edge,Safari entre outros.

Aplicações Web

São Soluções que são executadas diretamente no navegador, Possuindo a internet como meio de comunicação entre Client X Server não sendo necessário sua instalação.

Dispositivos Móveis

São eles, Celulares,Smartphones,Notebooks,Tablets,SmartWatchs.
